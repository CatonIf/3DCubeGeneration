Elbasan script keyboard


See the Wikipedia articles Elbasan alphabet and Elbasan Gospel Manuscript for clarification, and Robert Elsie's site for scans, translit and translations of the manuscript. If the characters appear as boxes, I suggest to install the Noto Sans Elbasan font.
Type in the box and it will convert the letters into the script. Differently from modern orthography, type ⟨n=⟩ for 𐔔, ⟨gh⟩ for 𐔥, ⟨kh⟩ for 𐔧, ⟨gh=⟩ for 𐔦. Please double-check the converted outcome with the manuscript scans, since it is not guaranteed to be accurate, e.g. ⟨nd⟩ is sometimes 𐔅, sometimes 𐔓𐔄 and sometimes 𐔔𐔄.
